Vanilla JavaScript + React Chatbot Widget

This demonstrates how to integrate the React Chatbot Widget into a vanilla JavaScript application using CDN links.

Once loaded, you'll see a floating chat icon in the bottom-right corner. Click it to start chatting!

Features Demonstrated:

  • Dynamic loading of React component in vanilla JS
  • CDN-based integration (no build process required)
  • Customizable theming and positioning
  • API key configuration
  • Error handling and status feedback
